## 컬러 사용 가이드

### 1. 배경 (Background)

| 변수 | 사용 목적 |
|------|-----------|
| `--color-bg-primary` | 페이지 기본 배경. 메인 콘텐츠 영역에 사용 |
| `--color-bg-secondary` | 섹션 구분용. 짝수/홀수 섹션을 번갈아 사용하여 시각적 리듬 생성 |
| `--color-bg-dark` | 히어로 섹션, 네비게이션 바, 임팩트가 필요한 특별 섹션 |

```css
/* 사용 예시 */
body { background-color: var(--color-bg-primary); }
.section-alt { background-color: var(--color-bg-secondary); }
.navbar { background-color: var(--color-bg-dark); }
```

### 2. 텍스트 (Typography)

| 변수 | 사용 목적 |
|------|-----------|
| `--color-text-primary` | 제목(h1~h3), 제품명, 핵심 메시지 |
| `--color-text-secondary` | 본문 텍스트, 부제목, 설명문 |
| `--color-text-muted` | 캡션, 날짜, 메타 정보, 푸터 텍스트 |

```css
/* 사용 예시 */
h1, h2, h3 { color: var(--color-text-primary); }
p { color: var(--color-text-secondary); }
.caption, .meta { color: var(--color-text-muted); }
```

### 3. 인터랙티브 요소 (Buttons & Links)

| 변수 | 사용 목적 |
|------|-----------|
| `--color-button-primary` | 주요 CTA 버튼 배경색 (구입하기, 시작하기) |
| `--color-button-hover` | 버튼 호버 상태 |
| `--color-button-outline` | 보조 버튼 테두리 (더 알아보기) |
| `--color-link` | 텍스트 링크 기본 색상 |

```css
/* 주요 CTA 버튼 */
.btn-primary {
  background-color: var(--color-button-primary);
  color: #FFFFFF;
  border: none;
  border-radius: 980px;
  padding: 0.6rem 1.4rem;
  font-size: 0.9rem;
}
.btn-primary:hover {
  background-color: var(--color-button-hover);
}

/* 보조 아웃라인 버튼 */
.btn-outline {
  background-color: transparent;
  color: var(--color-button-outline);
  border: 1px solid var(--color-button-outline);
  border-radius: 980px;
  padding: 0.6rem 1.4rem;
}

/* 텍스트 링크 */
a { color: var(--color-link); text-decoration: none; }
a:hover { color: var(--color-link-hover); text-decoration: underline; }
```

### 4. 테두리 & 구분선

| 변수 | 사용 목적 |
|------|-----------|
| `--color-border` | 카드 테두리, 입력 필드 보더 |
| `--color-border-light` | 섹션 구분선, 푸터 디바이더 |

```css
/* 사용 예시 */
.card { border: 1px solid var(--color-border); }
.divider { border-top: 1px solid var(--color-border-light); }
```

### 5. 액센트 & 시즌 컬러

| 변수 | 사용 목적 |
|------|-----------|
| `--color-accent` | 프로모션 배너, 특별 이벤트 강조, 배지, 알림 |

```css
/* 사용 예시 */
.badge-new { background-color: var(--color-accent); color: #FFFFFF; }
.promo-banner { background: linear-gradient(135deg, var(--color-accent), #FFD5D9); }
```

## 반응형 고려사항

### 화면 크기별 적용

```css
/* 모바일 (< 768px): 대비 강화, 터치 영역 확대 */
@media (max-width: 767px) {
  .btn-primary {
    padding: 0.8rem 1.6rem;
    font-size: 1rem;
    width: 100%;
  }
}

/* 태블릿 (768px ~ 1023px) */
@media (min-width: 768px) and (max-width: 1023px) {
  .btn-primary {
    padding: 0.7rem 1.5rem;
  }
}

/* 데스크톱 (>= 1024px): 기본값 유지 */
```

### 고대비 모드 (접근성)

```css
@media (prefers-contrast: high) {
  :root {
    --color-text-primary: #000000;
    --color-text-secondary: #333333;
    --color-text-muted: #555555;
    --color-border: #000000;
    --color-button-primary: #0051A3;
  }
}
```

### 모션 감소 설정

```css
@media (prefers-reduced-motion: reduce) {
  * {
    transition: none !important;
    animation: none !important;
  }
}
```

## 컬러 조합 가이드

### 권장 조합

| 배경 | 텍스트 | 용도 |
|------|--------|------|
| `--color-bg-primary` | `--color-text-primary` | 기본 콘텐츠 (대비율 15.4:1) |
| `--color-bg-primary` | `--color-text-secondary` | 본문 텍스트 (대비율 5.1:1) |
| `--color-bg-secondary` | `--color-text-primary` | 구분 섹션 콘텐츠 |
| `--color-bg-dark` | `#FFFFFF` | 히어로, 네비게이션 (대비율 21:1) |
| `--color-button-primary` | `#FFFFFF` | CTA 버튼 (대비율 4.6:1) |

### 금지 조합 (대비율 부족)

| 배경 | 텍스트 | 이유 |
|------|--------|------|
| `--color-bg-secondary` | `--color-text-muted` | 대비율 3.5:1 미만, WCAG AA 미달 |
| `--color-button-primary` | `--color-text-secondary` | 가독성 부족 |
